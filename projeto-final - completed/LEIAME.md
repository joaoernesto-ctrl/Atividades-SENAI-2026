BEM VINDO(A) AO MEU PROJETO FINAL!!!

Projeto com valor de 4 pontos do final da UC1 do SENAI, terceiro ano do ensino medio.

Esse projeto visa recriar de forma semelhante ou igualitaria a interface do Spotify. 
Foi utilizado conhecimentos dado em sala de aula sobre HTML, CSS e JS, aplicando-se em um único projeto sem o uso de Inteligência Artificial.
AS DIMENSÕES DA INTERFACE FORAM EFETUADAS EM COMPUTADORES DO SENAI, CASO A INTERFACE ESTEJA "QUEBRADA" FAVOR AJUSTE PARA QUE ELA FIQUE DE ACORDO COM O TAMANHO DO SEU MONITOR!
O site não possui adaptabilidade para mobile, pois o site base do Spotify também não possui este requisito.
